- 单例模式 singleton
- 策略模式 strategy
- 装饰模式 decorator

- 工厂方法模式 factory method
- 原型模式 prototype
- 外观模式  facade
- 建造者模式  builder
- 观察者模式  observer
- 抽象工厂模式 abstract factory
- 状态模式 state
- 适配器模式 adapter
- 桥接模式 bridge
- 命令模式 command
- 职责链模式 chain of responsibility
